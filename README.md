StarHunter
==========

This is a project for CooCox IDE and Visual Studio 2012 IDE.

CooCox part sets up STM32 Board to run as a Human Interface Device. STM32 is sending raw data from accelerometer through USB cable which are received and used as an input by StarHunter game.
Visual Studio part is a Snake like game, in which we are trying to collect as much stars as possible.


IDE and Compiler Setup
==========

To setup Visual Studio project you need to download Allegro 5 library and connect it to project.
